## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- **HTML** e **CSS** para estruturação e estilização moderna.
- **JavaScript** para a lógica de alternância de temas (Light/Dark mode).
- **Git** e **Github** para versionamento e hospedagem.

## 💻 Projeto

O **Links-Bio** é um agregador de links focado em um design limpo e funcional. Ele permite que o usuário centralize todas as suas redes sociais em uma única página, oferecendo uma experiência fluida com suporte nativo para temas claro e escuro.

## 🔖 Layout

O design foi inspirado em modelos de portfólio profissionais. A demonstração visual do layout pode ser conferida na imagem de preview acima, mostrando a adaptação tanto para Desktop quanto para Mobile.
